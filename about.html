ABOUT | WHAT IS THE SGDA

The Student Game Developer Alliance (SGDA) is a partnership between nine - and growing - student-led game development clubs from universities across California.

The SGDA was created with the intention to:

• PROVIDE valuable networking opportunities between students and professional developers.

• RAISE awareness for game development education,

• ENCOURAGE our member's passions for game development,

• ALLOW developers to mutually learn from each other’s approaches to game development.
